# Support Ticket Classification & Prioritization

## Project Overview

This project builds a Machine Learning model that automatically classifies customer support tickets into different ticket categories using Natural Language Processing (NLP).

## Dataset

customer_support_tickets.csv

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Machine Learning Workflow

- Load Dataset
- Data Exploration
- Text Preprocessing
- TF-IDF Vectorization
- Train-Test Split
- Model Training using Multinomial Naive Bayes
- Prediction
- Model Evaluation
- Classification Report
- Confusion Matrix

## Results

The model successfully classifies support tickets into their respective categories.
